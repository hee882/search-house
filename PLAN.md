# 프로젝트 개발 계획 (Roadmap)

## 🏁 Phase 1: MVP 개발 (진행 중)
- [x] 프로젝트 초기 스캐폴딩 (React + FastAPI)
- [x] LiveAnywhere 스타일의 지도 중심 UI 구현
- [x] 기본적인 경제적 입지 분석 알고리즘 (직선거리 기반)
- [x] 네이버 지도 API 연동 및 마커 표시
- [x] 네이버 부동산 매물 페이지 아웃링크 연결
- [x] 국토교통부 아파트 실거래가 공공데이터 API 연동
- [ ] **(Next)** 네이버 Directions 5 API를 활용한 실제 소요 시간 계산

## 🚀 Phase 2: 고도화 및 데이터 정교화
- [ ] 법정동/단지별 평균 월세 및 이자 비용 데이터베이스화
- [ ] 자차 연비 및 통행료 기반 이동 비용 계산 정교화
- [ ] 지하철 노선별 혼잡도 및 환승 횟수 가중치 적용 알고리즘
- [ ] 사용자 선호도 필터 (학군, 역세권, 숲세권 등)

## 💰 Phase 3: 수익화 및 배포

- [ ] SEO 최적화 (Next.js 전환 검토 또는 정적 페이지 생성)
- [ ] Kakao/Naver 소셜 로그인 및 관심 지역 저장 기능

## 🛠 아키텍처 가이드
- **환경 변수:** 로컬에서는 각 폴더의 `.env`를 사용하며, 절대 Git에 커밋하지 않음.
- **데이터 로직:** 복잡한 계산은 서버(`server/main.py`)에서 처리하고 프론트는 시각화에 집중.
- **UI 원칙:** 지도 영역을 최대화하고 사이드바는 세련된 절제미 유지.
