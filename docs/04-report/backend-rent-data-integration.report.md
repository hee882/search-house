# [Report] 스마트 주거 탐색 - 전월세 데이터 통합 및 UI/UX 개편

## 1. 프로젝트 요약
- **목적**: 기존 매매 실거래가 기반의 샐러리맨 워라밸 분석 서비스를 타겟 유저(사회초년생, 신혼부부)의 현실적 주거비에 맞춘 '전월세' 기반으로 고도화하고, 전반적인 UI/UX를 "스마트 주거 탐색" 브랜딩에 맞춰 전문적이고 직관적으로 개편했습니다.
- **주요 성과**:
  - 국토교통부 아파트 전월세 API 신규 연동 및 2만 9천여 건의 초기 데이터 확보
  - 전세/반전세/월세를 모두 '월 거주비용'으로 통합 환산하는 백엔드 엔진 고도화
  - 네이버 부동산 실매물 다이렉트 아웃링크 연결 (프롭테크 방식 채택)
  - 모바일 풀스크린 맵 및 바텀 시트 기반의 최신 지도 UX 적용
  - CORS 에러 및 Linter(Eslint) 에러 모두 해결

## 2. 주요 변경 사항 및 기술 스택
- **백엔드 (Python/FastAPI, SQLite)**:
  - `collector_rent.py`: 전월세 전용 수집 파이프라인 신규 구축.
  - `main.py`: `/api/optimize` 로직 개편 (월세/보증금 혼합 모델 적용, 응답 포맷팅).
  - DB 스키마: `rent_transactions` 테이블 추가 및 데이터 분리 저장.
- **프론트엔드 (React/Vite, TailwindCSS)**:
  - `App.jsx`: "매매" 토글 비활성화, 전월세 전용 데이터 렌더링 로직(보증금/월세 분리 텍스트), 네이버 부동산 아웃링크 로직 수정.
  - 브랜딩: 촌스러운 문구 제거, `Comprehensive Fatigue Model v1.0` 등 전문적인 카피 적용.
- **CI/CD (GitHub Actions)**:
  - `data_update.yml`에 전월세 수집(`collector_rent.py`) 스크립트 실행 스텝 추가.
