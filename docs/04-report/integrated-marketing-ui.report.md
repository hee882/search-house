# [Report] 주거비 위험도 시스템 및 마케팅 UX 통합 고도화

## 1. 개요 (Overview)
사용자의 경제적 상황을 직관적으로 보여주는 신호등 시스템과 서비스 바이럴을 위한 프리미엄 마케팅 요소들을 통합적으로 구축함.

## 2. 작업 내용 (Actions)
- **주거비 위험도 신호등:** 소득 대비 지출 비율에 따라 양호(20%), 주의(30%), 위험(40%) 색상을 적용하여 입력 및 결과 단계에서 직관적 인지 가능하도록 구현.
- **별칭(Alias) 검색:** '이수-총신대입구'와 같은 복수 명칭 역에 대해 관례적 별칭 검색 및 결과 병기 기능 추가.
- **철학 기반 HELP 가이드:** 단순 사용법을 넘어 '시간을 산다는 것'의 가치를 담은 스토리텔링 기반의 HELP 모달 구현.
- **프리미엄 OG 이미지 리뉴얼:** 공식 로고 반영 및 모바일 가독성 최적화된 v6 버전 제작 및 메타태그 적용.
- **2단계 바텀 시트 (Mobile):** 네이버/카카오 지도 앱과 유사한 Peek/Full/Hidden 3단계 드래그 바텀 시트 구현.

## 3. 주요 성과
- **직관성:** 사용자가 자신의 주거 예산 적정성을 신호등 색상으로 즉시 판단 가능.
- **바이럴 최적화:** SNS 공유 시 전문 서비스 느낌의 고품질 미리보기 카드 노출.
- **모바일 사용성:** 한 손 조작이 편리한 바텀 시트 인터랙션으로 모바일 앱 수준의 경험 제공.

## 4. 최종 확인
- 모든 UI 요소가 데스크탑 사이드바 토글 및 지도 패딩(Inset)과 완벽하게 호환됨.
- '이수역' 검색 시 '총신대입구역 (이수)'로 정상 노출 및 선택 확인.
