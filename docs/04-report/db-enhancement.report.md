# [Report] 실거래가 DB 다변화 및 통계 고도화

## 1. 프로젝트 요약
- **목적**: 기존 12컬럼 매매 DB를 24컬럼으로 확장하고, 신고가 자동 태깅 및 통계 API를 구축하여 부동산 시장 인사이트 플랫폼으로 진화
- **주요 성과**:
  - apt_seq 기반 아파트 고유 추적 체계 구축
  - 신고가 자동 판별 로직 (INSERT 시점 계산, 취소 거래 제외)
  - 기존 데이터 무손실 마이그레이션 (자동 감지 + 백업)
  - 통계 API 2종 (일별 거래량/신고가 추이)

## 2. 주요 변경 사항
- **collector.py**: 스키마 v2 마이그레이션, 14개 신규 필드 수집, check_new_high() 함수, backfill 로직, 인덱스 3개
- **main.py**: `/api/stats/transactions`, `/api/stats/new-highs` 엔드포인트 추가
