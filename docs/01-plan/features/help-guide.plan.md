# [Plan] HELP 가이드 — 사용법/개념/산식 모달

## Executive Summary

| 항목 | 내용 |
| ---- | ---- |
| Feature | help-guide |
| 시작일 | 2026-03-11 |
| 목표 | 상용 수준의 사용 가이드 모달 제공으로 서비스 이해도 및 신뢰도 향상 |

### Value Delivered

| 관점 | 내용 |
| ---- | ---- |
| **Problem** | 서비스 개념(기회비용, 피로도 가중치 등)이 생소하여 결과 해석이 어려움 |
| **Solution** | 헤더 HelpCircle 아이콘 → 풀스크린 모달로 5단계 사용법 + 산식 + 마케팅 콘텐츠 |
| **Function UX Effect** | 처음 방문한 사용자도 30초 내에 서비스 개념을 이해하고 결과를 해석 가능 |
| **Core Value** | 서비스 전환율 향상 + 바이럴 공유 시 설득력 확보 |

## 구현 상태

**이미 구현 완료** (2026-03-11 커밋 2a166e2):
- HelpModal 컴포넌트: 서비스 개념, 5단계 사용법, 비용 산식 4개, 주거비 한도 설명, 데이터 출처
- 헤더 HelpCircle 버튼 (PC/모바일 공통)
- 모바일: 바텀시트 형태, PC: 센터 모달
- 마케팅 슬로건: "집은 사는 곳이 아니라, 사는 방식입니다"

## 검증 방법

1. 헤더 ? 아이콘 클릭 → HELP 모달 정상 표시
2. 모바일에서 바텀시트 형태로 표시, 스크롤 가능
3. 외부 영역 클릭 시 모달 닫기
4. 산식 정보가 main.py의 실제 계산 로직과 일치하는지 확인
